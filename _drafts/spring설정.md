## 1. spring initializr?

프로젝트를 설정할때 항상 기입하는 항목에 대하여 실제 업무에선 어떠한 식으로 해당 항목을 인지하는지 확인해본다



## 2. Project metadata
- GroupId  
자신의 프로젝트를 식별해주는 고유 아이디  
- ArtifactId  
버전정보를 생략한 이름(jar), 일반적으로 프로젝트ID와 동일하게 작성한다
